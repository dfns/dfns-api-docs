
## ArchivePolicy
`RESTful Endpoint: delete /policies/{policyId}`

Deactivates a specific `Policy` entity by its `PolicyId` and marks its `status` as `Archived`.

### Input Query Parameters
* Path parameter `policyId`: `String`.
